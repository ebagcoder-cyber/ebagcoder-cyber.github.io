Navbar Tutorials
Here you can find a tutorial for a top navbar!
Step 1. Create a new Css file called something like navbar.css or Put this in your style sheet. Creating a new Css file would help keep your code organized.
Step 2. Start by creating a div class called ul
  
  ul { }
Step 3. Create a div class called li
  
  li { }
Step 4. Create a div class called li a
  
  li a { }
Step 6. Create a div class called li a:hover
  
  li a:hover { }
Step 7. Add style code to the div class ul
  
  ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
  overflow: hidden;
  background-color: gray;
}
Step 8. Add style code to the div class li
  
  li {
  float: left;
}
Step 9. Add style code to the div class li a
  
  li a {
  display: block;
  color: #ffffff;
  text-align: center;
  padding: 14px 16px;
  text-decoration: none;
}
Step 10. Add style code to the div class li a:hover
  
  li a:hover {
background-color: #C9C9C9;
}
